신 교과목 코드 읽는 방법

산업공학과의 새 교과목 코드는 IE. 뒤에 오는 네 자리 숫자에 과목의 학년·분야·성격·수강순서 정보가 담겨 있습니다. 각 자리의 의미는 아래와 같습니다.

학부 교과목 코드  IE.[학년][분야][일련][수강순서]

1학년

과목이 개설되는 표준 학년(1~4)을 나타냅니다.

11학년 22학년 33학년 44학년

2학문 분야

과목이 속한 9개 전공 분야를 나타냅니다.

0공통 1시스템설계 2시스템운영 3최적화 4확률모형 5데이터과학 6인간공학 7기술경영 8금융공학

3과목 구분 · 일련번호

분야 안에서의 순번이자, 과목의 난이도 성격을 나타냅니다.

1~4기초 과목 5~9심화 과목

4수강 순서

같은 분야 안에서 언제 들어야 하는지(선수 관계)를 나타냅니다.

0기초 · 선수과목 없음: 바로 수강 가능
1심화 ①: 동일 분야 0코드 과목을 이수한 뒤 수강
2심화 ②: 동일 분야 1코드 과목을 이수한 뒤 수강

대학원 교과목 코드  IE.G[분야][일련][성격]

앞자리 규칙(과정·분야)은 학부와 같고, 뒤 두 자리 의미가 다릅니다.

3일련번호 (학부 연계)

해당 대학원 과목과 연계되는 학부 과목이 있으면 그 학부 과목의 일련번호를 그대로 사용합니다. 연계 과목이 없으면 순서대로 1~9를 부여합니다.

제품공학 G150 ← 제품개발및품질설계 351

4과목 성격

대학원 과목의 수준을 기초 · 심화 · 응용으로 구분합니다.
